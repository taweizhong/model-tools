# Model-tools（模型工具）

## 简介

基于`go`语言 `GUI` 框架`Fyne`实现的模型辅助工具，主要功能如下：

1. 实现魔搭、`Hugging Face`等平台的LLM模型下载
2. 本地模型使用`git lfs`上传至`gitlab`仓库
3. 实现YOLO V11训练数据集的自定义格式划分

应用优势：

- 支持跨平台开发
- 免安装
- 标准信息流输出
- 简单高效

## 前提条件

确保应用环境已经安装`git lfs`

> [!NOTE]
>
> 注意事项：确保下载目录中没有和仓库同名的目录。
